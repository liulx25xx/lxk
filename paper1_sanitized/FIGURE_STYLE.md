# Active Figure System

The NeurIPS manuscript uses one restrained `paper_ocean` visual system.  This
file distinguishes active paper figures from older exploratory assets that
remain in the folder for provenance.

## Visual semantics

- Teal `#3C8FA4`: full/reported proxy, positive deltas, primary evidence.
- Coral `#C96F5F`: evaluator sensitivity, harmful transfer, negative deltas.
- Sand `#D8C58D`: secondary ablations.
- Gray `#7B8794`: neutral or minimal specifications.
- Ink `#202A33`: text and zero/reference lines.
- Grid `#DCE4E7`: quiet guides and borders.

Colors never carry the comparison alone: Figure 2 also uses filled circles,
hollow diamonds, squares, triangles, direct labels, and a zero line.  Figure 3
uses signed cell annotations and a diverging color bar centered at zero.

## Active figures

1. **Figure 1, structure-aware control view**: native TikZ in
   `main_neurips.tex`; its fills use the same teal/sand/coral semantics.
2. **Figure 2, evaluator sensitivity and negative transfer**:
   `figure2_experiment_summary.{pdf,svg,png}`, generated by
   `scripts/plot_experiment_summary.py` from
   `results/local_offline_experiments.json` and
   `results/paired_transfer_effects.csv`.
3. **Figure 3, cross-model heatmap**:
   `figure3_cross_model_heatmap.{pdf,svg,png}`, generated by
   `scripts/plot_cross_model_heatmap.py` from
   `results/cross_model_effects.csv`.

`scripts/paper_figure_style.py` centralizes palette, typography, vector-font,
and export settings for active Matplotlib figures.  Older `figure2_*` files are
not referenced by `main_neurips.tex`.

## Regeneration

```bash
python3 scripts/plot_experiment_summary.py
python3 scripts/plot_cross_model_heatmap.py
latexmk -pdf -interaction=nonstopmode -halt-on-error \
  -outdir=output/pdf main_neurips.tex
```

All active data figures are exported as vector PDF/SVG plus a 450 dpi PNG
preview.  The plotting scripts do not alter, filter, or selectively omit the
reported means or confidence intervals.
